# Проект: Платформа для фрилансеров FryLancer

## Project Overview
**FryLancer** — это экосистема инструментов для автоматизации работы фрилансеров. Текущий реализованный модуль: **Интерактивный бриф**. Этот инструмент позволяет фрилансерам быстро собирать требования у заказчиков и генерировать профессионально оформленные документы для начала работы.

## Tech Stack
- **Frontend**: React 19, Vite, TypeScript.
- **Styling**: Tailwind CSS (Custom Theme).
- **Export Libraries**: 
  - `jspdf` & `html2canvas`: Генерация PDF на основе визуального превью.
  - `docx`: Программная генерация документов Microsoft Word.
  - `file-saver`: Сохранение файлов в браузере.

## Features
1. **Интерактивная форма**: Сбор данных по ключевым разделам (Концепция, Детали, Условия).
2. **Экспорт в PDF**: Генерация документа с уникальным дизайном в стиле Linear (минимализм/бизнес).
3. **Экспорт в DOCX**: Создание редактируемого Word-файла со структурой заголовков.
4. **Экспорт в Markdown**: Быстрое сохранение текстового контента в MD-формате.

## Design Strategy
- **Тема**: Светлая (Light Mode).
- **Цветовая палитра**:
  - Фоновый цвет: `#fdf1e2` (кремовый/светлый беж).
  - Акцентный цвет: `#f97316` (насыщенный оранжевый).
  - Цвет текста: `#342626` (глубокий коричневый/черный).
- **Стиль интерфейса**: "Web App" (Linear/Stripe style) — крупные шрифты, тонкие линии, минималистичные карточки, фокус на типографике.

## Current Fields (App.tsx)
- **Project Name**: Название проекта.
- **Niche**: Ниша и описание бизнеса.
- **Goal**: Цель сайта.
- **Content**: Контент (информация).
- **Features**: Желаемый функционал.
- **Refs**: Референсы (что нравится).
- **Anti-Refs**: Анти-референсы (что не нравится).
- **Materials**: Ссылка на материалы.
- **Style**: Пожелания по стилю и цветам.
- **Budget**: Ориентировочный бюджет.
- **Deadline**: Желаемые сроки.
- **Contacts**: Контактные данные.
- **Time to Connect**: Удобное время для связи.
